Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
В выводе test() будет 2 т.к. возвращаемое значение именованное. При именованной переменной в возвращаемых defer обращается непосредственно к нему и имеет возможность его изменять.

В выводе anotherTest() будет 1 т.к. возвращаемая переменная неименованная. При неименованной возвращаемой переменной defer вносит изменения внутри своей области видимости {}.

defer используется для отложенного выполнения некоторого кода. Запускаются в обратном порядке из очереди до return'a, если их больше 1. Используют для отлова паник, закрытия каналов, разблокировки мьютексов. 

```
