Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
Вывод: ["3", "2", "3"]

Слайс это структура, состоящая из 3 полей: array - ссылка на родительский массив; len - длина слайс, cap - ёмкость слайса. 

На 11 строке инициализировали переменную и присвоили ей {"1", "2", "3"}. 
len = 3 cap = 3

На 12 передали слайс в функцию. *слайс передаётся по значению -> изменения внутри области видимости функции будут влиять вне области видимости этой функции.

На 17 строке заменили "1" на "3". Это изменение повлияет на исходный слайс.

На 18 строке добавили в слайс ещё одно значение.
len = 4 cap = 6 
*Т.к. после append'a выходим за ёмкость - происходит аллокация на новый родительский массив, в котором cap увеличивается побитовым сдвигом в 2 раза до 256, потом по формуле. Таким образом i на 18 строке ссылается на другой родительский массив -> последующие изменения на изначальный слайс никак не повлияют.
```
